# MirrorBody-X

AI-powered body analysis and fitness tracking application using computer vision for posture detection, body measurement, and movement analysis.

## Features

- Real-time body pose estimation
- Posture analysis with correction suggestions
- Body measurement estimation from camera feed
- Exercise repetition counting
- Fitness progress tracking over time

## Tech Stack

JavaScript, TensorFlow.js, MediaPipe, React, Canvas API

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
git clone https://github.com/karthik-idikuda/MirrorBody-X.git
cd MirrorBody-X
npm install
```

### Usage

```bash
npm start
```
